Parameter Scoping

Description

If the selected person is changed and this value is updated in the person service, this will not be reflected in the child component (PersonComponent).

This code doesn't work and that is the whole point of it. It is much better to use property and event binding to accomplish this. Better alternatives in some cases include services and ngrx. Trying to access variables by reference, as though they are a global, is not a good practice.
